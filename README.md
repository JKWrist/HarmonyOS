# HarmonyOS

## 介绍
鸿蒙系统应用基础开发代码，主要是学习鸿蒙开发

## 使用的开发语言

1.  Java
2.  JavaScript

## 开发环境

1.  Windows下开发
2.  IntelliJ IDEA Community Edition 2022.2
3.  DevEco Studio 3.0.0.991
4.  Visual Studio Code

## 参与贡献

1.  Fork 本仓库
2.  提交代码
3.  新建 Pull Request
4.  测试人员与评审人员至少有两个才可以代码入库

## 项目目标

1. 通过此项目了解鸿蒙开发
2. 学习Java、JavaScript等与之开发的基础知识
